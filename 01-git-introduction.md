# Git & GitHub Introduction

## What is Git?

- It is a Version Control System.
- It helps you to track changes made to files(source code).
- Helps developers to-
1. Track every changes & stores history.
2. Undo mistakes by going back to a previous version.
3. Work together without overwritig eachother's changes
4. Enables teamwork.
- It is free, open-source software & runs on your software.
- Git tracks only repositories , not your entire computer.

## What is GitHub?

- It is a web based platform to hosting Git Repositories & runs on clouds.(Online hosting for Git Repos).

## Repository/Repo:-

A Repo is a container that stores your project files & folders alongwith their change history.
